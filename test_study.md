# JAVA_study
====
### 상속

##### 부모클래스에 만들어진 필드, 메소드를 자식클래스가 물려받음
##### 부모의 생물학적 특성을 물려받는 유전과 유사
###### -> 자식클래스 간결

#### 상속의 개념
#####  부모클래스의 멤버(필드, 메서드, 이너클래스)를 자식 클래스가 내려받아(상속) 클래스 내부에 포함

#### 자바 상속의 특징
##### 부모 클래스: super class / 자식 클래스: sub class
 상속 횟수에 제한 없음
 자바에서 최상위 클래스는 object클래스, 모든 클래스는 object 클래스를 상속받음
